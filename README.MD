# EnumRTP
Многофункциональный плагин рандомных телепортов

## Использование
Основная команда:
> /rtp [safe | far | player | base]

## Основные преимущества
- Гибкая настройка GUI
- Удобное и простое управление телепортацией между мирами
- Оптимизация, выполненная в асинхронном формате
- Возможность создания собственных команд телепортации
- Четыре основных режима телепортации

## Требования
- Java 17
- WorldGuard
